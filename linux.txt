=> grep <= # is a command to search for matching patterns in a file
Example # grep (query) (pattern/s ..) file
-i # Ignore case sensitive
-v # Select the non-matching query
-n # Prefix each line with the line number in file
-w # Find the exact same
-c # count the number of matching query
. # Any chars after it
-r # Seach inside directory files  

You can also use regular expression in a specific appearance
grep fruit... -i -n -w fruits.txt

# You can also search for files and folders
ls | grep .py

# You can also search for keyword inside directory of files
grep -r -i hello files

=> CronJob <= 
is a linux program that allows to users to scheduling tasks to be
executed sometime in the future (Tasks often be shell scripts)
* * * * * (1=minute | 2=hour | 3=dOFm | 4=month | 5=dOFw)

* * * * * /home/script.sh or commands
To change later, run 'select-editor'.

=> crontab <=
Is a file containing the schedule of various cron entries that
should be run at specific time

crontab -l # List all cron jobs for the current user
crontab -e # Edit crontab entries (Add, Delete, Edit)
crontab -u username -l # List another user crons
crontab -u username -e # Edit another user crons

 m h dom mon dow
*/1 * * * * ls # Run every 1 minute
11 * * * * echo "Every 15" # Run when your minute include 15PM/15AM
1,10,15  * * * * echo "Every 1 10 15" # Multiple
@daily command # Another One
@reboot command # Another One
